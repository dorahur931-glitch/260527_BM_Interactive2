**1\.  Overview**

**1.1  Product Summary**

본 콘텐츠는 배달의민족 앱 내 팝업 웹뷰 형태로 제공되는 인터랙티브 이벤트 페이지입니다. 사용자는 "돌상" 위에 놓인 5가지 음식 중 하나를 선택하고, 선택한 음식이 상징하는 올해의 운세 키워드를 결과 페이지로 받아보는 구조입니다. 결과 확인 후 자연스럽게 미혼 한부모가정 캠페인 메시지와 연결되며, 돌잔치 방명록 이벤트 참여 시 배달 쿠폰이 지급됩니다.

**1.2  Goals**

•  배달의민족 "돌잡이" 콘셉트로 신규·기존 유저 인게이지먼트 증가

•  미혼 한부모가정 인식 제고 및 사회공헌 브랜드 이미지 강화

•  방명록 이벤트 참여 → 쿠폰 발급 → 실주문 전환 리텐션 효과

•  쿠폰 사용 시 기부금 자동 적립으로 가치 소비 경험 제공

•  소셜 공유를 통한 캠페인 바이럴 확산

**1.3  Non-Goals**

•  별도 앱 설치 또는 로그인이 필요한 외부 서비스 연동

•  실시간 다중 사용자 기능 (리더보드, 채팅 등)

•  음식 주문·결제 플로우의 직접 구현 (기존 배민 주문 플로우 활용)

**3.3  SCR-02: 돌상 메인 화면 — 상세 요건**

▌ 돌잡이 아이템 구성

| 음식 | 아이콘 | 상징 키워드 | 결과 카피 (요약) |
| ----- | ----- | ----- | ----- |
| 잔치국수 | 🍜 | 끈기 | "올해 당신은 끝까지 해낼 것입니다. 긴 면처럼, 끊어지지 않는 당신의 의지." |
| 치킨 | 🍗 | 인기 | "올해 당신의 주변은 당신으로 가득 찹니다. 치킨처럼, 모두가 원하는 존재." |
| 샐러드 | 🥗 | 건강 | "올해 당신의 몸과 마음이 빛납니다. 신선하고 가볍게, 건강한 한 해." |
| 비빔밥 | 🍲 | 계획 | "올해 당신은 모든 것을 제자리에 놓습니다. 재료 하나하나, 완성된 그림." |
| 피자 | 🍕 | 조화 | "올해 당신의 관계는 풍요롭습니다. 여럿이 나눌수록 더 맛있는 한 해." |

▌ 인터랙션 요건

•  음식 아이콘은 돌상 이미지 위에 오버레이. 각 아이템은 부드럽게 float 애니메이션 적용 (CSS keyframe or Lottie).

•  아이템 탭 시 확대 애니메이션 \+ 0.3s 딜레이 후 SCR-03 전환.

•  배경에 돌상(한국 전통 상보, 음식 소품) 일러스트 사용. 배민 캐릭터(배민이) 등장 가능.

•  첫 방문 유저: 5개 아이템 모두 선택 가능. 재방문 유저: 이전 선택 결과 그대로 표시 (재선택 불가).

•  화면 하단 "지금 골라봐\! 남은 올해의 운명은?" 유도 문구 노출.

**3.4  SCR-03: 결과 페이지 — 상세 요건**

▌ 결과 영역 (Above the fold)

•  선택 음식 대형 아이콘 \+ 이름 표시.

•  키워드 배지 (예: \# 끈기) — 배민 옐로우 또는 키컬러로 강조.

•  "남은 올해 당신이 이루게 될 한 가지" 대제목.

•  음식별 결과 카피 (3\~4줄 이내). 폰트: 배민 브랜드 폰트 또는 Noto Sans KR Bold.

•  공유 버튼 (카카오톡, 인스타그램 스토리, 링크 복사) 제공.

•  "다시 뽑기" 버튼: 선택 초기화 후 SCR-02 복귀 (1회 재시도 허용, 이후 비활성화).

▌ 캠페인 메시지 영역 (Below the fold — 스크롤)

•  섹션 구분선 \+ "그런데, 돌을 맞이하지 못한 아이들이 있어요." 브릿지 문구로 전환.

•  미혼 한부모가정 현황 카드: 통계 수치 \+ 실태 설명 (2\~3개 카드 슬라이드).

•  실제 한부모가정 사연 인용문 (익명 처리).

•  "첫 돌을 맞이하는 아이들에게 축하를 전해주세요" CTA → SCR-04 방명록 이동 버튼.

**3.5  SCR-04: 돌잔치 방명록 이벤트**

•  헤더: "🎉 첫 돌을 축하해요\! 방명록에 한 마디 남겨주세요"

•  텍스트 입력 필드: 최소 10자 / 최대 100자. 실시간 글자 수 카운터 표시.

•  닉네임 입력 필드: 최대 10자 (선택 입력, 미입력 시 "익명" 처리).

•  제출 버튼 탭 시 — 로딩 인디케이터 → 쿠폰 발급 성공 메시지 → SCR-05 이동.

•  이미 참여한 유저(쿠폰 발급 완료)는 입력 폼 대신 "이미 참여하셨어요 🎁" 상태 표시.

•  방명록 하단에 다른 유저가 남긴 메시지 타임라인 노출 (최신 10개, 더보기 가능).

**3.6  SCR-05: 쿠폰 발급 완료**

•  쿠폰 코드 카드 UI: 코드 표시 \+ "복사" 버튼.

•  "지금 주문하러 가기" 버튼 → 배민 홈 또는 특정 카테고리 딥링크 이동.

•  "공유하기" 버튼: 카카오톡/인스타 스토리로 이벤트 링크 \+ 결과 카드 이미지 공유.

•  하단 안내: "쿠폰 사용 시 한 끼 식사 지원 기금이 미혼 한부모가정에 전달됩니다."

•  쿠폰 유효기간 및 사용 조건 명시 (예: 발급 후 7일, 최소 주문금액 조건 등).

**5\.  UI / UX Design Specification**

**5.1  Design Principles**

•  배달의민족 브랜드 정체성(한국적 유머·따뜻함)과 전통 돌잡이 문화를 결합.

•  "돌상" 일러스트는 픽셀아트 또는 플랫 일러스트 스타일, 배민 캐릭터 아이덴티티 반영.

•  인터랙션은 가볍고 즐거워야 하며, 캠페인 메시지로의 전환은 자연스럽고 강요적이지 않게.

•  결과 페이지는 SNS 공유를 전제로 한 카드형 레이아웃 (OG 이미지 자동 생성).

**5.2  Color & Typography**

| Primary Color | \#FFD600 — 배민 옐로우 |
| :---- | :---- |
| **Secondary Color** | \#0EEED3 — 캠페인 포인트 컬러 (청록) |
| **Background** | \#FFFDF4 — 따뜻한 아이보리 |
| **Text Primary** | \#1A1A1A |
| **Text Secondary** | \#666666 |
| **Title Font** | 배민 주아체 또는 Noto Sans KR Black |
| **Body Font** | Noto Sans KR Regular / Bold |

**5.3  Animation Spec**

•  돌상 등장: 아래에서 위로 슬라이드 인 (0.5s, ease-out).

•  음식 아이콘 idle: 위아래 3px float loop (2s, ease-in-out infinite).

•  아이템 선택: 탭 → scale(1.25) → scale(1) → 0.3s 페이드 아웃 → 결과 화면 전환.

•  결과 카드 진입: 스크롤 기반 fade-in-up (IntersectionObserver).

•  쿠폰 발급: 컨페티(confetti) 파티클 이펙트 1.5s.

  created\_at:     ISO8601,  
  updated\_at:     ISO8601  
}

▌ GuestbookEntry

{  
  entry\_id:   string,     // UUID

  created\_at: ISO8601  
}
